# 🌧️ Rainfall Prediction using Machine Learning
## 📌 Project Overview
This project predicts whether it will rain or not based on weather conditions using Machine Learning. The model is trained on historical weather data and performs binary classification (Rain / No Rain).

---

## 🤖 Machine Learning Model
- Algorithm Used: Random Forest Classifier
- Type: Classification
- Output: Yes / No (Rain prediction)

---

## 📊 Features Used
. Temperature
. Humidity
. Wind speed
. Pressure
. Other weather parameters (based on dataset)
## ⚙️ Steps Followed
- Data collection
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training using Random Forest
- Model evaluation

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

---

## 🎯 Result
The model successfully predicts whether rainfall will occur based on input weather features.
